About Me

Hi, I’m Sauravi Rai

Product leader building agentic AI systems — from blueprint to production.

I spend my days designing and shipping AI agents that do real work — not demos. My recent focus is on composable multi-agent architectures: small, specialized agents that cooperate around a shared memory and identity model, with humans always in the loop. The pieces in this portfolio are my own attempts to build, document, and stress-test that idea.

Behind the scenes

How this portfolio was built

🎯

Curated, not exhaustive

From 27 internal share-pages + dozens of local artifacts, 9 made the cut. Every piece had to be either generalizable, technically interesting, or methodologically novel — not just a status update.

🛡️

Multi-pass sanitization

~550 substitutions across the 9 files: real names → role placeholders, corporate emails redacted, internal vendor/system names genericized. Verified zero leaks across 7 categories.

📦

Zero build dependencies

Every piece is a single self-contained HTML file (or HTML + co-located assets/). No webpack, no npm install — just open in any browser.

♿

WCAG 2.2 Level AA

This index page ships with skip-to-content, visible focus rings, semantic landmarks, sufficient color contrast, and proper alt-text on every thumbnail.
