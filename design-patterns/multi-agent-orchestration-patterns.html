What you’re looking at

Three real composition patterns from a five-agent personal stack — how the agents share context, query each other, and stay loosely coupled. Documented as a reference (closer to an ADR than a tutorial).

The five agents

🚪

Room Agent

Books meeting rooms; reads shared preference profile

🗒️

Task Wrangler

Pulls Outlook, Jira, Confluence into a daily digest

🏷️

Org Change Agent

Owns stakeholder maps, OKRs, re-org context

📊

AI Usage Tracker

Pulls live adoption data; renders disposable HTML reports

👁️

Stack Shepherd

The meta-agent — tends the stack itself: updates docs, pushes to git, re-invokes stale agents
