# sauravirai.github.io

🌐 **Live site:** https://sauravirai.github.io

Personal portfolio for **Sauravi Rai** — Senior Director, Product Management.
22 years across Walmart, Amazon, Microsoft, Nordstrom, and Oracle. Bengaluru-based, open to opportunities.

## Stack

- Single-page static site — `index.html`
- [Tailwind CSS](https://tailwindcss.com) (via CDN)
- Inter / Inter Tight (Google Fonts)
- Auto-deployed via GitHub Pages from the `main` branch root

## Local preview

```bash
python3 -m http.server 8000
# open http://localhost:8000
```

## Drop-in résumé

Drop a `resume.pdf` next to `index.html` to make the "Download résumé" link work.
